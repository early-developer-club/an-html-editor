# 쇼핑몰 상세 페이지 에디터

비개발자도 사용하기 쉬운 시맨틱 HTML 상세 페이지 에디터

## 기술 스택

- **프레임워크**: Vite + React
- **언어**: TypeScript
- **스타일링**: CSS

## 프로젝트 구조

```
src/
├── components/    # React 컴포넌트
├── hooks/         # 커스텀 React hooks
├── stores/        # 상태 관리
├── types/         # TypeScript 타입 정의
├── utils/         # 유틸리티 함수
└── styles/        # 스타일 파일
```

## 개발 시작

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 린트 검사
npm run lint

# 코드 포맷팅
npm run format
```
